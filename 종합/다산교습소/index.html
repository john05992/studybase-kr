다산 교습소
마지막으로 반복적인 리듬과 친근한 말투를 사용해 학습 내용을 자연스럽게 반복하도록 유도함으로써, 학생이 학습에 중독되는 효과를 기대한다. 다산 교습소은 이 학생의 경우, 매 수업마다 오답 점검 시 어디서부터 틀렸는지 추적하며 취약점을 집중 보완하는 시스템을 운영했는데, 이는 단순한 반복이 아닌 ‘정밀 진단 기반’의 학습이었다. 다산 교습소은 예를 들어, 한 지문을 20분 넘게 소요하는 경우, 그 원인이 어휘 부족인지 구조 분석 미숙인지 세밀하게 분석할 수 있어, 맞춤형 보완이 가능해진다. 예를 들어 한 문항에서 ‘의미’ 3점, ‘근거’ 2점, ‘표현력’ 1점으로 나뉠 경우, 학생은 각 단계별로 어떤 요소가 포함되어야 하는지 명확히 인지하게 됩니다. 기출문제 분석 후에는 각 문항의 출제 빈도와 유형 흐름을 매핑하여, 기존 교과서 순서를 벗어나 고빈도 주제를 먼저 다루는 식으로 진도를 재조정한다. 둘레 구하기나, 복습을 위한 메모를 간단하게, 반복해서 보기 쉽게 만드는 것도 도움이 됩니다. 학생들은 학습 과정을 시작할 때 흔히 일관된 실천률을 유지하기 어려운 상황에 직면한다는 점을 전문가가 객관적으로 지적한다.
